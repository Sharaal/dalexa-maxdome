# Features

## Implemented

* Flash Briefing Skill API
  * Aktueller Tipp des Tages im JSON Format
* Request/Response: 
  * "Alexa, frag maxdome was es Neues gibt"
  * "Alexa, frag maxdome nach dem Tipp des Tages"
* Slots: 
  * "Alexa, frag maxdome was es Neues an {genre} gibt"
